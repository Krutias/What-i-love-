# What I Love Project
This is my project for the Nexus Tutorial, where we were asked to create something about what we love. I chose to write about BTS, because their music, message, and performances mean a lot to me.
 I wanted to share a little bit of the love i have for them through this project
✨ Features
-A collection of my favorite BTS photos and albums
-Highlighting favorite songs and why they inspire me
-Clean and easy-to-browse grid layout
-Contact form for fans to share their thoughts.

📂   project Files
-index.html – main page
-about.html – information about BTS
-gallery.html – photo gallery of BTS
-contact.html – contact form for fans
-css/ – all styling files
-images/ – BTS photos

 Notes 
I've used HTML for structuring the pages and CSS for styling,colors and layout. Each page is designed with BTS inspired colors.
